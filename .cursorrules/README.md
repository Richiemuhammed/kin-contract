# Cursor Rules

This folder contains project-specific rules and conventions for Cursor IDE.

## Files

- **naming-conventions.md** - File, folder, and code naming conventions (kebab-case preference)

## Purpose

These rules help maintain consistency across the codebase and guide Cursor AI to follow project conventions when:
- Creating new files
- Suggesting code
- Refactoring
- Generating code

## Usage

Cursor IDE will automatically reference these rules when assisting with code generation and suggestions.
